# claude-code-webui (sugyan) — archived

Web chat interface for the Claude Code CLI (Deno/Node backend + React frontend, streaming
responses). MIT-licensed, so legally fine — but the repo was **archived in May 2026**, so it is
excluded as a foundation for client work.

Community Docker images exist (`heywoodlh/claude-code-webui` — adds Nix, rootless Podman,
optional nginx basic auth) and are a reasonable *reference* for containerising a UI + CLI pair.

Same verdict for **textcortex/claude-code-sandbox** (320★, no licence, archived) — it pioneered
the "Claude Code in Docker with browser terminal + auto git branch" UX in 2025 and is worth
reading for design history, nothing more. Also **omnara** (2,650★, Apache-2.0, archived 2026-01)
— mobile command-centre for agents; archived, skip.
