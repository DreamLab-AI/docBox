# crystal

MIT-licensed **Electron desktop app** for running multiple Claude Code sessions in parallel git
worktrees, with diff review, run scripts, and session persistence. Included for completeness:
it demonstrates the multi-session/worktree UX well, but it is a desktop app, not a web
interface, and worktrees-not-containers for isolation — so it does not fit the single-purpose
web container model directly. Mine it for UX patterns (session rehydration, diff-first review).
